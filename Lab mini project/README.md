# Moving Traffic Scene Project

![Project Output](https://github.com/kmajhi/Computer-Graphics-Lab/blob/main/assets/output.gif)

A simple 2D animated mini project built with **OpenGL** and **GLUT** for the **Computer Graphics Lab**.


## Project Overview
This project creates a colorful outdoor road scene with smooth animation. It includes a moving car, a moving truck, animated clouds, trees, bushes, flowers, a road light, the sun, and a roadside environment.

The scene is drawn using basic OpenGL primitives such as polygons and custom circle-based shapes. Animation is handled by updating object positions continuously inside the idle function.

## Features
- 2D animated traffic scene
- Moving car and truck
- Animated clouds
- Trees, bushes, flowers, and road light
- Sun and sky background
- Keyboard interaction for cloud movement
- Exit option using `Esc`, `q`, or `Q`

## Technologies Used
- C++
- OpenGL
- GLUT

## Controls
- **Left Arrow** → move clouds left
- **Right Arrow** → move clouds right
- **Esc / q / Q** → exit the program



## How to Run
1. Open the project in Code::Blocks or any C++ IDE with OpenGL/GLUT support.
2. Make sure GLUT is properly configured.
3. Build and run the program.

## File
- `Moving Traffic Scene.cpp` → main source code for the project
